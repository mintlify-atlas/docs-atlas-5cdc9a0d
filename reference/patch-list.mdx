Cromite applies **332 patches** to Chromium to enhance privacy, security, and functionality. This page lists all patches organized by category.

<Note>
For detailed information about each patch, including the full code changes, visit the [Cromite patches directory](https://github.com/uazo/cromite/tree/master/build/patches) on GitHub.
</Note>
