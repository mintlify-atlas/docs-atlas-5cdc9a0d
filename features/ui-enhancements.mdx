### URL Display

<Info>
**Disable omission of URL elements** - Cromite shows the full URL without hiding "trivial" subdomains.
</Info>

Many browsers hide parts of URLs (like "www" or "m"), which can be exploited for phishing. Cromite shows complete URLs for security.

### Search Engine Settings

<Note>
Android version includes desktop UI for search engine settings, providing better control over search engine configuration.
</Note>

**Enhanced search engines:**
- Google search engine that forces English language
- Disables RLZ and field experiment parameters
- DuckDuckGo Lite added as option

## Certificate Display

**EV Certificates appear as normal** - Extended Validation certificates don't get special UI treatment

<Note>
EV certificates don't provide additional security, so Cromite doesn't give users a false sense of extra security.
</Note>
